# Crypto Cipher Encyclopedia

Crypto Cipher Encyclopedia is a Rust console application that presents historical and modern cryptographic ciphers in a searchable offline catalog.

The project focuses on explaining how classical ciphers work, when they appeared and what eventually replaced them.

## Features

- Offline cipher catalog
- Search by name
- Browse historical timeline
- Security classification
- Modular architecture

## Included Ciphers

- Caesar Cipher
- Vigenère Cipher
- Playfair Cipher
- Enigma
- Hill Cipher
- One-Time Pad
